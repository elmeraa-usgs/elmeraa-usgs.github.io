#30DayChartChallenge

What is the #30DayChartChallenge?

As stated on the site, “The #30DayChartChallenge is a community-driven challenge that happens each year during April. Every day, there is a certain prompt, grouped into five categories: comparisons, distributions, relationships, time series, and uncertainties. Anyone is welcome to contribute, no matter which data source or tool is used to create the visualizations.” More info on code of conduct from creators here.
